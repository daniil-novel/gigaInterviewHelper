# Changelog

Все заметные изменения проекта `gigaInterviewHelper` по датам.

## 2026-04-05

### Added
- Добавлен полноценный Gmail OAuth 2.0 flow с `authorization code`, `refresh_token` и чтением писем через Gmail API.
- Добавлена диагностика Gmail OAuth и понятные сообщения об ошибках Google API с прямой ссылкой на включение Gmail API.
- Добавлен автоматический и ручной запуск сессии из входящих писем через кнопку `Начать сессию`.
- Добавлен блок статистики по авто-режиму: приглашения, авто-сессии, вопросы, подтверждённые ответы и текущая очередь.
- Добавлен `CHANGELOG.md` для фиксации изменений по дням.

### Changed
- Основной пользовательский поток упрощён до сценария: письмо -> сессия -> открыть AI-интервью в браузере -> вставить ответ вручную.
- Интерфейс переведён на тёмную визуальную систему с более сильным контрастом и адаптивной сеткой.
- Дашборд перестроен в табовый layout: `Сессии`, `Входящие письма`, `Профиль`, `Настройки`.
- Навигация после действий стала контекстной: сохранения и проверки возвращают пользователя в соответствующую вкладку.
- Продвинутая Telegram-интеграция понижена до optional advanced-режима и убрана из основного сценария.

### Fixed
- Исправлен баг с отсутствующим импортом `parsedate_to_datetime` в Gmail-сервисе.
- Исправлены raw JSON-ответы в браузере: Gmail-проверки и почтовая синхронизация теперь показывают понятные UI-уведомления.
- Улучшен парсинг ссылок из HTML-писем: отбрасываются технические URL вроде `w3.org`, а приоритет получают реальные interview/invite/chat ссылки.
- Исправлены проблемы адаптивности интерфейса, из-за которых action-кнопки и боковые блоки ломали layout на плотных экранах.

### Repository
- Подготовлен `.gitignore` для безопасной публикации проекта.
- Проект оформлен и опубликован в GitHub-репозиторий `gigaInterviewHelper`.
